# dot-claude

개인 Claude 설정을 공유하는 레포지토리입니다.

이 레포지토리는 Claude와 함께 작업할 때 사용하는 커스텀 규칙과 베스트 프랙티스를 관리합니다.

## Table of Contents

### Rules

- **[Next.js Best Practices](./rules/nextjs-rules.md)**
  - Next.js 특화 API 사용 규칙 (`next/dynamic`, `next/server`, RSC, optimizePackageImports 등)

- **[React Best Practices](./rules/react-rules.md)**
  - React 코어 API 사용 규칙 (hooks, memo, Suspense 등)

## 사용 방법

이 레포지토리의 파일들을 로컬의 `.claude` 혹은 프로젝트의 `.claude` 폴더로 옮겨 사용할 수 있습니다.
